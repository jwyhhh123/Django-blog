About Me

This is the website of Wenyue Jin. I am a computer science student in the University of Birmingham.

I appreciate the opportunity that the univesity provides me rich resources and knowledge from many different fields. Currently I am in my second year study and I have learnt:

Year 1:

  • Mathematical Foundations: Contents include and not only include: numbers, mathematical principles of bases, analytic geometry, matrices, set, relations... and regular languages, automata, P and NP... These are quite struggling for human beings.

  • Logic: Propositional logic, natural deduction(proofs), predicate calculus... and isomorphism, lambda calculus, curry-howard correspondence. This module is linked to second year module(functional programming).

  • Data Structure: Array, Iteration, Recursion, List(stack, queue, doubly linked list..), Searching(linear, binary), Efficiency and complexity, Trees, Binary search treeï¼ŒBinary heap tree, Sorting, Hash table, graphs(path problem).

  • Java: I was learning java basics, objective-oriented programming and some stuff of network connection and threads.

  • Artificial Intelligence: Theoretically I learnt the concepts ofSwarm Intelligence, depth/width-first search, A-star, Hill climbing, Simulated Annealing, k-Nearest Neighbours, naive bayes, linear/logistic regression, neural network... And I was also engaging in building EV3 robot to complete tasks like path finding and maze exploring.

Year 2:

  • Functional programming: This module focuses on teaching Haskell(built-in functions, monad, pureness and laziness...)

  • Mathematical modeling and decision making: In the first half of the module(probablistic modelling), it includes principle component analysis and guassian distribution(1 to many); in the second half, it includes something I DON'T understand(Discrete fourier transfrom) and introduction of Makov chain.

  • Security and Networking: This is a huge module as it covers a lot about encryption/decryption methods, such as AES, RSA, and some techniques of capturing traffic from the network(wireshark,burp). It also includes hacking in acess control system and other protocal based communication, website SQL injection and other web attack(steal cookie, XSS...). Additionally, a brief explanation of reverse engineering was introduced.

  • Software Engineering: Mainly about software developing architecture based on UML.

  • System programming and C/C++: From architecture level, it introduced what a computer is and how a computer system works(kernel, user space, processes). From hardware level, it introduced how CPU, RAM, disk and other components cooperate and how operating system handles different tasks in different situations. Beyond this, C language was taught based on these. (C++ was briefly introduced).

To be continued.
